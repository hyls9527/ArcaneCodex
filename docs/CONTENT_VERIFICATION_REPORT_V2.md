## 四、核查方法论

### 4.1 判定标准

| 严重程度 | 判定标准 |
|----------|----------|
| 严重 | 可能误导用户对 AI 能力或安全性的认知 |
| 中等 | 技术实现与描述不符 |
| 轻微 | 无依据的评级或过度美化 |

### 4.2 修正原则

1. **保持核心意图**: 不删除功能，仅修正不实描述
2. **客观描述**: 用"生成标签"替代"理解画面"
3. **标注限制**: 说明功能依赖用户配置
4. **删除无依据内容**: 移除无法验证的评级
