## 四、核查方法论

### 4.1 判定标准

| 严重程度 | 判定标准 | 示例 |
|----------|----------|------|
| 严重 | 产品定位或竞品对比存在虚假声明 | "第一个"、"行业首创"、"超越所有" |
| 中等 | 完成度或评分虚高 | "100%"、评分过高 |
| 轻微 | 技术细节不准确 | 覆盖率概念误用 |

### 4.2 修正原则

1. **删除绝对化表述**: "第一个"、"所有"、"超越" → 改为客观描述
2. **标注实现状态**: "行业首创" → "探索性功能"或"设计阶段"
3. **修正数学错误**: "排除后100%" → 正确计算值
4. **降低虚高评分**: 根据实际功能调整评分
