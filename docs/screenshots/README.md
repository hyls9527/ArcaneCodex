# Screenshots

This directory contains screenshots for the README and documentation.

## Required Screenshots

Please add the following screenshots to showcase the application:

1. **dashboard.png** - Main dashboard view showing statistics and charts
2. **gallery.png** - Gallery view with image grid
3. **search.png** - Semantic search in action
4. **settings.png** - Settings page showing AI configuration

## Screenshot Guidelines

- Use 1920x1080 resolution or higher
- Use sample images (not personal photos)
- Show the app in both light and dark themes
- Highlight key features

## Adding Screenshots

1. Take screenshots using the app with sample data
2. Resize to reasonable dimensions (max 1200px width for README)
3. Optimize file size (use PNG for UI, JPEG for photos)
4. Update README.md to reference the screenshots
